## 🇧🇷 Português

> Um motor de processamento de imagens e visão computacional focado na correção automatizada de provas e gabaritos de múltipla escolha.

Este projeto não é apenas um script de leitura visual, mas uma aplicação robusta construída sob os princípios da **Clean Architecture (Ports and Adapters)**. Projetado para rodar em servidores e corrigir milhares de provas em lote, o sistema utiliza álgebra linear e processamento de matrizes através do **OpenCV**, mantendo o domínio de negócio em **Java 21** e **Spring Boot 3** completamente isolado das ferramentas de infraestrutura.

## 🇺🇸 English

> A computer vision and image processing engine focused on the automated grading of multiple-choice tests and answer sheets.

This project is not just a visual reading script, but a robust application built on the principles of **Clean Architecture (Ports and Adapters)**. Designed to run on servers and batch-grade thousands of tests, the system uses linear algebra and matrix processing via **OpenCV**, while keeping the **Java 21** and **Spring Boot 3** business domain completely isolated from infrastructure tools.
